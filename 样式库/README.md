# 样式库

可复用的 UI 样式与效果参考，**不参与** webpack 打包（无 `index.ts` 入口）。开发前端界面或状态栏时，从此处复制、改编样式与组件片段。

## 目录

| 子目录 | 来源 | 内容 |
|--------|------|------|
| [found1](./found1/) | 原 `util/found1` | 5 个参考 UI Vue 单文件组件（`参考UI1.vue` … `参考UI5.vue`） |
| [案例](./案例/) | 原 `src/案例` | 按钮、卡片、进度条、输入框、Tooltips 等 Vue/HTML/CSS 效果片段 |

## 使用方式

1. 在浏览器中打开 `.html` 文件预览静态效果（如 `案例/打字效果/typing.html`）。
2. 将 `.vue` 中的 `<template>` / `<style>` 复制到目标项目的组件中，按需改类名与变量。
3. 注释中若引用了本库路径，请使用 `样式库/案例/...` 或 `样式库/found1/...`。

## 与其它目录的关系

- **`示例/`** — 可运行的完整前端界面/脚本示例（会打包到 `dist/`）。
- **`备份工作区/`** — 暂停中的完整项目（ADven、Goth 等）；Goth 专属参考仍在 `备份工作区/Goth/参考/`。
- **`样式库/`** — 零散的样式与 UI 片段，供复制参考，不单独部署。
