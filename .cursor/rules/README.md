# Rules (canonical)

`.cursor/rules` is the **canonical source** for project Agent rules (`.mdc`).

The following IDE folders are **mirrors** synced from here (same basename, `.md` extension):

- `.augment/rules/`
- `.roo/rules/`
- `.kilocode/rules/`
- `.clinerules/` (flat; no `rules/` subfolder)

When rules change, update files in this directory first, then re-sync the mirrors.
